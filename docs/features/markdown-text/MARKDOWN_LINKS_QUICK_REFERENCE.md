# Markdown Links - Quick Reference Card

## Basic Syntax

```
[Link Text](URL)
```

## Common Examples

### External Website

```
Visit [our website](https://example.com)
```

### Internal Page

```
Read our [Terms](/tnc)
```

### Email

```
Contact [support@example.com](mailto:support@example.com)
```

### Phone

```
Call [+60 12-345 6789](tel:+60123456789)
```

## Multiple Links

```
Buy from [Shopee](https://shopee.com.my) or [Lazada](https://lazada.com.my)
```

## Where to Use

✅ **Entry Method & Submission** - Main field with tip
✅ **Prizes** - Link to prize details
✅ **Eligible Stores** - Link to store locators
✅ **Winners Information** - Link to announcements

## Tips

- Keep link text descriptive: ✅ "contest form" ❌ "click here"
- Test links before publishing
- Use internal links (`/page`) when linking within JomContest
- Use external links (`https://...`) for external websites

## Common Mistakes

❌ Missing brackets: `[Link Text(https://example.com)`
❌ Missing parentheses: `[Link Text]https://example.com`
❌ Space in URL: `[Link](https://example .com)` - use `%20` instead
❌ Unmatched brackets: `[Link Text](https://example.com`

✅ Correct: `[Link Text](https://example.com)`
